# instadraw
## O instaDraw é um BOT que marca todas as pessoas de sua lista de (Seguindo) em uma foto especifica.

### 1º: instale o pacote chamado Selenium com o seguinte comando no PowerShell
![2020-05-04 01_36_34-Window](https://user-images.githubusercontent.com/35803016/81182334-088af180-8f84-11ea-9fb7-6962ffe9d4f9.png)


### 2º: baixe o GeckoDriver no seguinte repositorio: https://github.com/mozilla/geckodriver/releases

![2020-05-04 01_37_44-](https://user-images.githubusercontent.com/35803016/81182692-8818c080-8f84-11ea-8c6f-a5706b6c56fc.png)


### 3º: Crie uma pasta no Disco Local C: chamada: geckodriver

![2020-05-04 01_39_36-Window](https://user-images.githubusercontent.com/35803016/81182956-e6de3a00-8f84-11ea-8737-8346554fccd8.png)

### 4º extraia o conteudo baixado no parte 2º para dentro da pasta criada no disco local C:

![2020-05-04 01_43_21-Window](https://user-images.githubusercontent.com/35803016/81183715-de3a3380-8f85-11ea-8e0d-80ca65c271ef.png)

### 5º altere o conteudo das 3 variaveis (seuusuario,suasenha,linkdafoto) e execute o terminal.

#### seuusuario: digite o seu login para entrar no instagram
#### suasenha: digite a senha para entrar no instagram
#### linkdafoto: digite o link da foto que será comentada.
